\subsection{Detailed Description}
C++ interface This is a C++ wrapper of the C A\+P\+I above adding multi process safety and a few extra functions. 

The multi process safety only works within processes using this interface. This means that two or more processes using this interface can only use each of the G\+P\+I\+Os one process at the time. If another process uses the C interface or accesses the device memory directly it will conflict with the process using this A\+P\+I.
