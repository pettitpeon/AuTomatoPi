Detailed Description

GPIOs APIs.

C API:

  • The C API is thread-safe unless explicitly specified otherwise.
  • The C API has an initializations counter. Do not call the HwGpioExit() function without having called HwGpioInit() first.
  • The C API is not multi-process safe. Processes can overwrite each other

C++ API
The object oriented C++ API is multiprocess safe between processes using this API

TBaBoolRC HwGpioSetClkHWPWM ( double  frequency,
uint32_t  resolution,
EHwGpioHWPWMMode  mode

Set the frequency and resolution of the hardware PWM.

The clock frequency is 19.2 MHz The maximum resolution is 4095 steps (12-bit) Please choose these wisely and bare in mind that the actual frequency is an estimate because only positive clock frequency divisors can be used. freq_max @ min resolution (1-step) is 9.6 MHz freq_max @ max resolution (12-bit) is 2.34 kHz Examples: freq_max @ 1024-steps (10-bit) is 9.375 kHz freq_max @ 256-steps (8-bit) is 37.5 kHz Conditions: frequency * resolution * 2 < 19.2 MHz resolution <= 4095

Returns
Error or success
Parameters
[in]frequencydesired frequency in Hz
[in]resolutiondesired resolution in steps
[in]modePWM mode
