virtual void IBaLog::GetLogInfo ( TBaLogInfo *  pInfo)
pure virtual

Gets the information and options.

The const char pointers of the TBaLogInfo struct must not be changed or mangled with. They point to internal memory. Also they invalidated when the logger instance is destroyed. Accessing the freed memory results in undefined behavior.
