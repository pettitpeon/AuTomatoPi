const char* HwCom1WRdAsync ( const char *  serNo,
TBaCoreMonTStampUs *  pTs 
)

Read the value of the 1W sensor asynchronously.

This works with an internal worker thread. The serNo must have the form:
XX-XXXXXXXXXXXX [devFam]-[devID]

Returns
On success, the contents of the sensor file, otherwise 0
Parameters
[in]serNoOptional serial number of the sensor eg: "28-0215c2c4bcff". If null, the first sensor with family ID 28 is used
[out]pTsoptional timestamp of the reading
