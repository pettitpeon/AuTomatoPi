Detailed Description

Inter-process communication module.

It implements an IPC server-client solution with remote procedure calls (RPCs) and remote variable calls

◆ OsIpcSvrRunning()

TBaBoolRC OsIpcSvrRunning ( )

Get the server status.

Returns only if the server started or stopped successfully, and no heart beat of any kind.

Returns
True if the server thread started running, false if stopped.

◆ OsIpcExitSvr()

TBaBoolRC OsIpcExitSvr ( )

Sends exit signal to the IPC server and waits with a timeout for it to stop.

If it timeouts, there is a small memory leak, but the thread exits when it frees itself.

Returns
True if success, otherwise, false
